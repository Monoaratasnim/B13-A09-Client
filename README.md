# EduQueue – Tutor Booking System

🔗 **Live Website:**
EduQueue Live Site: https://mediqueue-eta.vercel.app

## 📌 Project Overview

EduQueue is a modern tutor booking platform where students can easily browse tutors, book online learning sessions, and manage their scheduled classes efficiently. The system helps eliminate manual scheduling issues and prevents time slot conflicts by providing an organized digital booking experience.

This project was developed using **Next.js**, **Express.js**, **MongoDB**, and **JWT Authentication**.

---

## 🚀 Features

* 🔐 Secure user authentication using JWT
* 👨‍🏫 Browse available tutors by Name and Date
* 📅 Book online tutor sessions with preferred time slots
* 🎟️ Generate digital session tokens for each booking
* ⚡ Prevent duplicate or conflicting bookings automatically
* 📱 Fully responsive design for mobile, tablet, and desktop
* 🧾 Students can manage and track their booked sessions
* 🌐 Fast and optimized frontend built with Next.js
